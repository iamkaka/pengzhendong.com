# 人生浪费指南 — 使用指南

## 如何添加新文章

### 1. 创建文件

在 `public/content/posts/` 下创建一个 `.md` 文件。文件名用英文或拼音（不要用中文），比如 `my-first-trip.md`。

### 2. 编写内容

文件顶部必须有一段 frontmatter（头部信息），用 `---` 包裹，然后写正文。

```markdown
---
title: "文章标题"
date: "2026-06-28"
tags: ["随笔"]
excerpt: "一句话摘要，会显示在首页和文章列表中"
---

# 文章标题

在这里写正文，支持所有 Markdown 语法……

## 小标题

### 三级标题

正文段落。

- 列表项
- 列表项

> 引用文字

**加粗** *斜体* `代码`

[链接文字](https://example.com)

![图片描述](图片URL)
```

### 3. 保存文件

保存后**刷新浏览器**，文章会自动出现在首页和文章列表中，无需重启服务器。

### 关键规则

| 项目 | 说明 |
|------|------|
| 文件位置 | `public/content/posts/xxx.md` |
| 文件编码 | UTF-8（VS Code 右下角可查看） |
| 文件名 | 英文或拼音，不要中文 |
| title | 文章标题，用双引号包围 |
| date | 格式 `YYYY-MM-DD`，用双引号包围 |
| tags | JSON 数组格式，如 `["随笔", "游记"]` |
| excerpt | 一句话摘要 |
| --- | 顶部和底部各一行三个横线 |

### 标签系统

在 tags 里写中文标签，多个标签用逗号分隔：

```
tags: ["游记", "云南"]
```

首页的「标签」页面会自动按标签分类所有文章。

## 如何添加新照片

### 文件位置

照片数据在 `src/data/photos.ts` 中。打开这个文件，在 `photos` 数组里添加一条记录：

```typescript
{
  id: 6,                                          // 递增的 ID
  url: 'https://images.unsplash.com/...',          // 照片 URL 或本地路径
  title: '你的照片标题',
  location: '拍摄地点',
  date: '2026-06-28',                              // 格式 YYYY-MM-DD
  description: '对这张照片的描述，可以写感受、拍摄背景、当时的故事等……',
},
```

### 照片 URL 说明

三种方式提供照片：

1. **外部链接**：直接贴图片直链 URL
2. **本地文件**：把图片放到 `public/images/` 目录，URL 填 `/images/your-photo.jpg`
3. **Unsplash**：示例使用了 Unsplash 免费图片

### 照片详情页

点击任意照片会跳转到独立详情页，展示：
- 大图
- 拍摄日期和地点
- 照片描述
- ← → 切换上一张 / 下一张

## 栏目结构

```
导航：  首页  →  标签  →  摄影  →  关于

首页：  3 篇最新文章 + 3 张最新照片
标签页：按标签分类浏览所有文章
摄影页：照片画廊，点击进入详情
关于页：博客介绍
```

## 启动方式

```bash
cd blog
npm run dev
```

浏览器访问 `http://localhost:5173`

构建静态文件：

```bash
npm run build     # 产物在 dist/ 目录
npm run preview   # 本地预览构建产物
```
